# 前言

欢迎来到本Spring Boot洗衣店订单管理系统项目！此项目是一款基于Java开发的实战项目，适用于毕业设计或学习交流。在这里，你将获得完整的源码、文档报告以及详细的代码讲解。

# 内容介绍

本项目是一款功能完善的洗衣店订单管理系统，主要包括用户管理、订单管理、洗衣服务管理、库存管理等功能模块。通过使用Spring Boot框架，以及MySQL数据库，实现了高效、稳定的数据处理。此外，项目还使用了JS、Vue和CSS3等前端技术，使得系统界面更加美观、易用。

# 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16

# 核心代码

以下是本项目的一段核心代码，展示了如何使用Spring Boot和MySQL实现用户查询：

```java
// UserController.java
@RestController
@RequestMapping("/user")
public class UserController {

    @Autowired
    private UserService userService;

    @GetMapping("/list")
    public List<User> list() {
        return userService.list();
    }
}
```
